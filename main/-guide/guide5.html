    믹스인 안내

    @include width($width1, $width2);
    // width1: 가로값, width2: 최대 가로값 (무조건 가운데정렬)
     
    @include height($height, $bdwidth, $bdcolor);
    // height: 전체 세로값, bdwidth: 선 두께, bdcolor: 선 색
     
    @include ellipsis($lines: 1, $line-height);
    // lines: 보일 라인 수, line-height: 글자 줄간격(세로값)
     
    @include bg2($type);
    // type1: 넘치게, type2: 꽉 차게
     
    @include position($direction);
    // left: 가로 가운데 정렬, top: 세로 가운데 정렬, center: 가로세로 가운데 정렬, none: 정렬 해제(position:relative)
     
    @include hidden($width, $height);
    // width: 감춰질 요소의 가로값, height: 갑춰질 요소의 세로값
     
    @include visible($width, $height);
    // width: 보여질 요소의 가로값, height: 보여질 요소의 세로값
     
    @include icon($type, $content);
    // content: 아이콘의 유니코드, type1: inline-block, type2: absolute, type3: 영역 전체를 채움
     
    @include square($size);
    // size: 가로 세로 사이즈 (정사각형)
     
    @include hover($target);
    // before: 회전할 가상선택자가 before, after: 회전할 가상선택자가 after
     
    @include title($margin, $font, $color);
    // margin: 요소의 전체 마진(위, 아래, 왼쪽, 오른쪽 모두), font: 글자 크기, color: 글자 색
     
    @include bul($padding);
    // padding: 왼쪽 패딩값(블릿 모양은 before로 직접 지정해야 함)
     
    @include list($type, $mr, $mt, $ea);
    // mr: 가로 마진, mt: 세로 마진, ea: 한 줄에 나올 갯수, type1: 리스트가 모두 한 줄로, type2: 리스트 줄바꿈
     
    @include btn($height, $padding, $bdwidth, $bdcolor, $bgcolor, $font, $color);
    // height: 버튼 세로값, padding: 버튼 패딩, bdwidth: 선 두께, bdcolor: 선 색, bgcolor: 배경 색, font: 글자 크기, color: 글자 색 (오버시 스타일도 저장되어있음)

    작성 규칙

    네이밍 규칙

    • 스네이크 표기법(underbar_nate), 한자리수 번호 (예 : .main_wrap, .section1)
    • 한 개의 단어로 구성된 경우 영문 소문자 작성을 기본으로 하며 숫자는 맨 앞에 나올 수 없다.
    • 언더바(_)는 2개 이상의 단어 조합시 사용한다. 예) header_wrap
    • 단어와 숫자 조합시 언더바(_)를 생략한다. 예) group1

    기타

    • 스타일 작성 시 가상선택자에 텍스트 정보 제공하지 않는다. (접근성 문제)
    • a 요소 안에 블록 요소 있을 경우 a에 무조건 display: block (접근성 문제)
    • 폼 요소를 커스텀 할 경우 :focus 스타일이 별도로 존재해야 한다. (접근성 문제)
    • table caption에 position:absolute 속성 사용하지 않는다. (사파리에서 셀 가로값을 조정할 수 없어짐)
    • 콘텐츠 페이지 마크업 시 메뉴 넘버로 클래스명 주지 않는다. 예) sub010101 (메뉴 순서가 변경될 경우 혼란을 야기할 수 있음)
